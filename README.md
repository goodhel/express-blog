# Backend Blog
Backend Blog with Express JS and MongoDB

## Installation
1. Install depedencies with yarn or npm
2. Setting .env like .env-example and change the database url with your own
3. Run service
```
# Development
    - yarn dev
# Production
    - yarn build
    - yarn start
```
